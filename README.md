# 📚 电子书数据集 (Ebook Dataset)

本仓库提供从 **[山海作品推荐](https://welearn.shanyaohaicuo.com/)** 整理导出的电子书资源数据集。  
包含电子书元数据（标题、分类标签、网盘链接），覆盖文学、哲学、心理学、教材教辅、备考资料等 20+ 个类别。

> 🎯 **公益声明**：本项目为纯公益性质，旨在促进知识传播与教育资源共享，帮助学习者快速发现优质电子书资源。项目不收取任何费用，不接受商业赞助，所有元数据对公众完全开放。
> 📌 **技术说明**：本仓库仅存储资源的**元数据**（标题、标签、链接），不存储任何实体电子书文件。所有链接均指向第三方网盘（百度网盘/夸克网盘），请用户自行判断链接有效性及版权合规性。

## ✨ 核心特性

- **📖 资源丰富**：涵盖经典文学、思想哲学、心理学、教材教辅、备考资料等多个领域。
- **🔗 一键获取**：每个资源标题均直接链接至网盘（百度网盘或夸克网盘），点击即可访问（部分链接含提取码）。
- **🏷️ 智能标签**：通过标签体系实现资源的精准分类与快速定位。
- **📅 持续维护**：不定期从源站同步新资源，并修正失效链接。
- **📝 专业规范**：采用 JSON 结构化元数据格式，支持自动化的资源索引与校验；分类体系参照图书馆学科分类法设计。
